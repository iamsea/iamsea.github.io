Hexo基本配置与使用

发表于 2017-11-16 | 分类于 Hexo

修改主题

  1. 下载主题，将主题文件夹放入themes文件夹
  2. 修改_config.yml里面的theme字段为新的主题文件夹名
  3. 如果该主题有相关依赖插件，记得安装
  4. 提交Github完成安装

修改导航栏

这里以next主题为例
初始导航栏只有首页跟归档

  1. 进入主题文件夹
  2. 打开_config.yml主题配置文件

  3. 修改menu字段，取消about(关于)及tags(标签)的注释

  4. 修改后博客会多出对应的导航项，但导航页面要另外添加

新建博客

比如新建一篇博文叫《Hexo安装》

hexo-install也可以用Hexo安装，但建议不要用中文
此时会在source目录下的_posts目录生成hexo-install.md文件
打开这个文件

进行如下编辑

title: Hexo安装是博文标题
tags: Hexo是标签信息
categories: Hexo是分类信息

添加用户头像

将头像图片(eg: avatar.png)放在next主题source/images目录下
取消主题配置文件_config.yml下的avatar字段的注释
