域名等级

顶级域名

.com, .cn, .org等

一级域名

baidu.com, google.com等

二级域名

www.baidu.com, music.baidu.com, mail.google.com等。www.baidu.com是特殊的二级域名

以此类推

xxx.xxx.baidu.com是四级域名

注：一级域名在添加域名解析的时候主机记录用@表示或者置空。
